### 🖊️ 编辑操作

- **Ctrl + D** → 选中下一个相同单词
- **Alt + Shift + ↑/↓** → 向上/下复制一行
- **Ctrl + /** → 注释/取消注释当前行
- **Ctrl + Shift + K** → 删除当前行
- **Ctrl + Shift + \\** → 跳转到匹配的括号

### 📂 文件与窗口

- **Ctrl + P** → 快速打开文件
- **Ctrl + Shift + N** → 新建窗口
- **Ctrl + Shift + W** → 关闭窗口
- **Ctrl + Tab** → 切换打开的文件
- **Ctrl + Shift + T** → 重新打开最近关闭的文件

### 🔍 搜索与替换

- **Ctrl + F** → 查找
- **Ctrl + H** → 查找并替换
- **F3** → 查找下一个
- **Shift + F3** → 查找上一个
- **Alt + Enter** → 选中所有匹配项

### 🖥️ 终端与调试

- **Ctrl + `** → 打开内置终端
- **Ctrl + Shift + `** → 新建终端
- **F5** → 开始调试
- **Shift + F5** → 停止调试
- **F9** → 设置/取消断点
- **F10** → 单步跳过
- **F11** → 单步进入
