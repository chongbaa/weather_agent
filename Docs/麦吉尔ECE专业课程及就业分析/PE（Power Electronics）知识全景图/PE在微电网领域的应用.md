# 微电网（Microgrid）

就是把电力电子学的“四层金字塔”直接搬到实际战场里打一场综合仗，缺了哪一层你都跑不起来。

学电力电子 = 直接能干微电网。

|微电网核心组成部分|直接用到的电力电子知识（金字塔层级）|典型器件/拓扑/控制|
|---|---|---|
|光伏并网/离网逆变器|第1层 SiC/GaN + 第2层 三电平NPC、T-type + 第3层 SVPWM + 第4层 光伏MPPT|800V SiC MOSFET、三电平逆变、MPPT算法|
|储能双向PCS（电池变流器）|第1层 SiC MOSFET + 第2层 双向Buck-Boost/双有源桥DAB + 第3层 恒功率/恒压/恒流/VSG虚拟同步机控制|1200V SiC模块、双向T型三电平、DQ控制+VSG|
|柴油/燃气发电机并联接口|第2层 整流+DC/DC + 第3层 下垂控制/虚拟同步机|晶闸管整流或维也纳整流器、下垂控制|
|微电网主控能量管理系统（MGCC）|第3层 数字控制（多机并联、功率均分、无缝切换） + 第4层 孤岛检测、黑启动、低电压穿越|DSP/FPGA实现多环控制、IEC 61850通信|
|关键负载不间断供电（UPS功能）|第2层 在线互动/双向逆变 + 第3层 超快无缝切换控制|双向逆变器、相位快速锁相、<10ms切换|
|静止同步补偿器（STATCOM功能）|第2层 多电平MMC或T型级联 + 第3层 瞬时功率理论 + 第4层 无功/谐波补偿|低压MMC、dq变换、重复控制+有源滤波|
|固态开关/固态断路器|第1层 SiC MOSFET/GaN + 第2层 超快固态开关拓扑 + 第3层 纳秒级过流检测|1200V/50A SiC全控固态断路器|
|微电网内部DC母线（可选）|第2层 多端口DC/DC + 第3层 母线电压下垂均压控制|多端口隔离DC/DC、母线电压协调控制|

# 总结

微电网就是“PE技术的大集成+大练兵”。

你现在学的每一章，在真正的微电网项目里都会直接出现，甚至很多公司直接把“能否独立设计一个100kW储能PCS或光伏逆变器”作为微电网方向研究生的入门考核。

更狠的现实：

2025年以后，国内所有211/985高校的微电网实验室，能做出来的最高水平，基本就等于学生电力电子功底的天花板。

所以你现在背的那张金字塔图，到了微电网方向，直接就是项目总师的思维导图。
