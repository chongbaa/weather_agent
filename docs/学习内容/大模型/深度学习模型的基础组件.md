# 📘 **现代深度学习模型的基础组件（完整体系）**

我把它们分成 6 大类，每类都给你：

- 组件名称
    
- 作用
    
- 典型出现位置
    
- 为什么重要
    

你可以按这个顺序逐个学习。

# 🧩 **一类：核心数学组件（所有模型的底层）**

## **1. Linear（全连接层）**

- **作用**：线性变换
    
- **出现**：所有模型
    
- **重要性**：所有注意力、MLP 都由它构成
    

## **2. LayerNorm / RMSNorm**

- **作用**：稳定训练
    
- **出现**：Transformer、扩散模型
    
- **重要性**：没有 Norm，Transformer 根本训练不动
    

## **3. 激活函数（GELU / SiLU / ReLU）**

- **作用**：非线性
    
- **出现**：MLP、UNet
    
- **重要性**：决定模型表达能力
    

# 🧠 **二类：Transformer 组件（LLM / CLIP / Vision Transformer 的核心）**

## **4. Self-Attention（自注意力）**

- **作用**：建模序列内部关系
    
- **出现**：LLM、ViT、CLIP
    
- **重要性**：Transformer 的灵魂
    

## **5. Cross-Attention（交叉注意力）**

- **作用**：跨模态信息融合
    
- **出现**：SD、SDXL、CLIP、图文模型
    
- **重要性**：图生文、文生图的关键
    

## **6. Multi-Head Attention（多头注意力）**

- **作用**：多视角理解信息
    
- **出现**：所有 Transformer
    
- **重要性**：提升表达能力
    

## **7. MLP Block（前馈网络）**

- **作用**：非线性变换
    
- **出现**：所有 Transformer
    
- **重要性**：占 Transformer 参数量的 70%
    

## **8. Positional Encoding（位置编码）**

- **作用**：提供顺序信息
    
- **出现**：LLM、ViT、音频模型
    
- **重要性**：没有位置编码，模型不知道顺序
    

# 🎨 **三类：扩散模型组件（Stable Diffusion / SDXL 的核心）**

## **9. UNet（U 型网络）**

- **作用**：预测噪声
    
- **出现**：SD1.5、SDXL、ControlNet
    
- **重要性**：扩散模型的主干
    

## **10. Timestep Embedding（时间步嵌入）**

- **作用**：告诉模型当前扩散步骤
    
- **出现**：所有扩散模型
    
- **重要性**：扩散过程的核心条件
    

## **11. Scheduler（采样器）**

- **作用**：控制去噪过程
    
- **出现**：DDIM、Euler、DPM++
    
- **重要性**：决定生成质量和速度
    

## **12. VAE（变分自编码器）**

- **作用**：图像压缩/解压
    
- **出现**：SD、SDXL
    
- **重要性**：把 1024×1024 压成 latent
    

## **13. Text Encoder（文本编码器）**

- **作用**：把 prompt 转成 embedding
    
- **出现**：CLIP、OpenCLIP、T5
    
- **重要性**：决定 prompt 理解能力
    

# 🎥 **四类：视频模型组件（Video Diffusion / Sora 类模型）**

## **14. Temporal Attention（时间注意力）**

- **作用**：建模帧之间的关系
    
- **出现**：Video Diffusion、TimeSformer
    
- **重要性**：视频的关键
    

## **15. 3D Convolution（3D 卷积）**

- **作用**：同时处理空间 + 时间
    
- **出现**：视频分类、视频生成
    
- **重要性**：捕捉运动信息
    

## **16. Motion Module（运动模块）**

- **作用**：建模运动一致性
    
- **出现**：视频扩散模型
    
- **重要性**：让视频不抖、不乱动
    

# 🖼️ **五类：图像模型组件（CNN / ViT）**

## **17. Convolution（卷积）**

- **作用**：局部特征提取
    
- **出现**：ResNet、UNet
    
- **重要性**：图像模型的基础
    

## **18. Patch Embedding（图像切块）**

- **作用**：把图像切成 patch
    
- **出现**：ViT、CLIP
    
- **重要性**：让图像能进 Transformer
    

## **19. Residual Block（残差块）**

- **作用**：稳定深层网络
    
- **出现**：ResNet、UNet、Transformer
    
- **重要性**：深度学习革命的关键
    

# 🔊 **六类：多模态组件（图文、音频、视频统一模型）**

## **20. Projection Layer（投影层）**

- **作用**：把不同模态映射到同一空间
    
- **出现**：CLIP、Qwen-VL、LLaVA
    
- **重要性**：多模态对齐的核心
    

## **21. Modality Encoder（模态编码器）**

- **作用**：图像/音频/视频编码
    
- **出现**：CLIP、Whisper、VideoMAE
    
- **重要性**：多模态输入的入口
    

## **22. Fusion Module（融合模块）**

- **作用**：融合不同模态
    
- **出现**：多模态 LLM
    
- **重要性**：让模型“理解世界”
